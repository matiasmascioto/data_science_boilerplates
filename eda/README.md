# EDA Boilerplate
Boilerplate to quickly get started with an *Exploratory Data Analysis* project in *Jupyter Notebook*.

## Dependencies
*  [Jupyter Notebook](https://jupyter.org/)
*  [pandas](https://pandas.pydata.org/)
*  [seaborn](https://seaborn.pydata.org/)


## File structure
```
|-- data
	|-- titanic.csv
|-- notebooks
	|-- eda.ipynb
```
